# 🎬 CineBox — Seus Filmes Favoritos

![Capa CineBox](capa-cinebox.png)

Bem-vindo ao **CineBox**, um projeto de interface web inspirado no layout do YouTube, desenvolvido para exibir uma coleção de filmes de forma organizada e elegante.

## 🚀 Funcionalidades

- **Layout Estilo YouTube:** Interface familiar com barra de busca, categorias e grade de vídeos.
- **Player Integrado:** Suporte a trailers via Iframe do YouTube (exemplo: Debi e Loide).
- **Filtros Inteligentes:** Navegue por gêneros como Comédia, Ação, Drama, entre outros.
- **Busca em Tempo Real:** Encontre seus filmes favoritos instantaneamente.
- **Design Responsivo:** Totalmente adaptado para dispositivos móveis e desktop.
- **Tema Dark:** Experiência visual confortável e moderna.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica.
- **CSS3:** Estilização avançada com Flexbox, Grid e variáveis.
- **JavaScript (Vanilla):** Lógica de filtragem, busca e manipulação do DOM.
- **Font Awesome:** Ícones modernos.
- **Google Fonts:** Tipografia Roboto.

## 📂 Como Usar

1. Clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Aproveite a experiência CineBox!
